# Installation
1. Install GNU stow, and any other packages in the .config (kitty, i3gaps, etc)
2. Clone into home
3. cd into cloned directory 
4. run `stow .`
## ZSH PACKAGES TO INSTALL:
`zsh-syntax-highlighting`
## Wallpaper chaning

2. Configure it
